Install Stealth at the command prompt if you haven't yet:

  ```
  gem install stealth
  ```

At the command prompt, create a new Stealth bot:

  ```
  stealth new mybot
  ```

Change directory to `mybot` and start the server:

  ```
  cd mybot
  stealth server
  ```
